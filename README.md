# Lista de Tarefas Vue com Temporizador

Este é um aplicativo simples em Vue que implementa uma lista de tarefas com um temporizador para cada tarefa. O aplicativo permite que os usuários criem tarefas, iniciem e parem os temporizadores associados a cada tarefa, marcando as tarefas como concluídas.

## Recursos

- Adicionar tarefas com uma descrição
- Iniciar e parar temporizadores para cada tarefa para rastrear o tempo gasto nelas
- Exibir o tempo total gasto em cada tarefa e sua descrição
- Design responsivo para dispositivos móveis e desktop
- Dark e Light mode com uso do tailwindCSS

## Instalação

1. Clone o repositório:
   git clone https://github.com/MessiasViana/TaskTimerManager

2. Navegue até o diretório do projeto:
   cd vue-lista-de-tarefas-com-temporizador

3. Instale as dependências:
   npm install

## Uso

1. Inicie o servidor de desenvolvimento:
   npm run serve

2. Abra o navegador e visite o link de localhost indicado no terminal para acessar o aplicativo.

3. Use as seguintes funcionalidades no aplicativo:

- Adicione uma nova tarefa inserindo uma descrição
- Clique no botão "Iniciar" ao lado de uma tarefa para iniciar o temporizador para aquela tarefa.
- Clique no botão "Parar" para pausar o temporizador de uma tarefa.
- Clique no botão "Concluir" para concluir a tarefa e adicionar ela na lista.
- O tempo total gasto em cada tarefa é exibido ao lado da descrição da tarefa.

## Tecnologias Utilizadas

- Vue.js - Framework JavaScript para construção de interfaces de usuário
- Vuex - Padrão de gerenciamento de estado e biblioteca para aplicativos Vue.js
- Vue Router - Roteador oficial para aplicativos Vue.js
- TailwindCSS - Framework CSS para desenvolvimento web responsivo e mobile-first
